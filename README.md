To-Do List App (Tkinter)

A simple desktop To-Do List application built using Python and Tkinter. It allows users to add, remove, and manage daily tasks easily.

Features
Add new tasks
Remove selected task
Delete all tasks (with confirmation)
Simple and clean GUI

Simple Calculator (Python)

A basic command-line calculator built using Python. It allows users to perform simple arithmetic operations like addition, subtraction, multiplication, and division.

Features
Addition
Subtraction
Multiplication
Division (with zero check)
